# ArgentBank

### OpenClassrooms - Projet 13

Application web permettant aux clients d'Argent Bank de se connecter et de gérer leurs comptes et profils.

## Objectifs

### Authentification des utilisateurs
1. Mettre en place l'application et intégrer les maquettes
2. Mettre en place le state global avec Redux et la connexion de l'utilisateur
3. Modifier le nom d'utilisateur via le formulaire

### Gestion des transactions
1. Analyser les maquettes
2. Écrire les différents endpoints dans le fichier Swagger
3. Préparer les livrables

## Prérequis

- [Node.js](https://nodejs.org/en/)
- [Yarn](https://yarnpkg.com/)

## Installation

### 1 - Application Front-end

```bash
git clone https://github.com/Weeskin/DA-P13-ArgentBank-PS.git
cd DA-P13-ArgentBank-PS
yarn install
yarn start
```

### 2 - API

```bash
git clone https://github.com/OpenClassrooms-Student-Center/Project-10-Bank-API
```

Suivre les instructions du README de l'API.
L'API tourne sur [http://localhost:3001/](http://localhost:3001/)
